-- SUMMARY --

The Drupal curlypage module allows you to add pagepeel, "magic corners"  type
banners to your website. 

For a full description of the module, visit the project page:
  http://drupal.org/project/curlypage

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/curlypage


-- REQUIREMENTS --

None.


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.

* If you want to upgrade from pageear follow instructions in
  INSTALL.TXT


-- CONFIGURATION --

* Configure user permissions in Administer >> User management >> Permissions >>
  curlypage module:

  - administer curlypages

    Users in roles with the "administer curlypages" permission will be able to
    administer curlypages.

  - use PHP for curlypage visibility

    Users in roles with the "use PHP for curlypage visibility" permission will
    be able to configure curlypages visibility using PHP code.

* Configure, add or delete your curlypages in Administer >> Site building >>
  Curlypages.


-- TROUBLESHOOTING --

* If you find any problem, write an issue in the project issue queu:

  - http://drupal.org/project/issues/curlypage
